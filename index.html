Auxilium Advanced Inventory

The ultimate upgrade for Game Creator 2! This powerful extension transforms the standard Inventory Module into a professional, RPG-style, grid-based system packed with features modern games demand: a unified hotbar, equipment panel, advanced sorting, category filtering, right-click context menus, and more.

  • Unified Equipment Panel A modern "direct slot" system for seamless drag-and-drop gearing.
  • Action-Ready Hotbar Fully functional hotbar with customizable keyboard bindings for instant actions.
  • Context-Aware Menus Intelligent right-click menus that only show relevant actions for any item.
  • Powerful Sorting One-click sorting by name, weight, or price that protects your hotbar slots.
  • Category Filtering Filter your inventory by item type using tabs or a dropdown menu.
  • And much more... Smart placement, safety drop system, and dynamic UI displays.
